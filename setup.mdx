## Microsoft Office Add-ins

### For Regular Users

1. Open Microsoft Excel, Word, or Outlook
2. Go to the **Insert** tab
3. Click **Get Add-ins** or **Office Add-ins**
4. Search for "Passionfruit"
5. Click **Add** to install

Alternatively, you can download the add-ins directly:
- [Excel Add-in](https://appsource.microsoft.com/passionfruit-excel)
- [Word Add-in](https://appsource.microsoft.com/passionfruit-word)
- [Outlook Add-in](https://appsource.microsoft.com/passionfruit-outlook)

### For Administrators

To deploy Passionfruit add-ins across your organization:

1. Download the manifest files:
   - [Excel Manifest](https://static.passionfruit.earth/manifests/excel-manifest.xml)
   - [Word Manifest](https://static.passionfruit.earth/manifests/word-manifest.xml)
   - [Outlook Manifest](https://static.passionfruit.earth/manifests/outlook-manifest.xml)

2. Deploy using Microsoft 365 Admin Center:
   - Navigate to **Settings** > **Integrated apps**
   - Click **Upload custom apps**
   - Select the manifest file
   - Follow the deployment wizard

3. Configure deployment settings:
   - Choose target users/groups
   - Set permissions
   - Enable automatic updates

### Troubleshooting Microsoft Add-ins

If you encounter issues:

1. Verify Office version compatibility
   - Requires Microsoft 365 or Office 2019+
   - Must be updated to latest version

2. Check internet connectivity
   - Add-ins require internet access
   - Verify corporate firewall settings

3. Clear Office cache
   - Close all Office applications
   - Delete contents of `%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\`
   - Restart Office application

## Browser Extensions

### Installation

1. Choose your browser:
   - [Chrome Web Store](https://chrome.google.com/webstore/passionfruit)
   - [Firefox Add-ons](https://addons.mozilla.org/passionfruit)
   - [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/passionfruit)

2. Click **Add to Browser** or **Install**

3. Follow browser prompts to complete installation

### Enterprise Deployment

For enterprise-wide deployment:

1. Download the extension package:
   - [Enterprise MSI Installer](https://static.passionfruit.earth/enterprise/passionfruit-extension.msi)

2. Deploy using Group Policy:
   ```
   msiexec /i passionfruit-extension.msi /quiet
   ```

3. Configure extension policies through your MDM solution

### Troubleshooting Browser Extensions

Common solutions:

1. Update your browser to the latest version

2. Clear browser cache and cookies:
   - Settings > Privacy and Security
   - Clear browsing data
   - Restart browser

3. Check extension permissions:
   - Manage extensions
   - Verify Passionfruit permissions
