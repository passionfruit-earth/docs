## Chrome Extension

### Installation

1. Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/passionfruit/fpkjoohpmifejbbeincijlocdgadmbin)
2. Click "Add to Chrome"
3. Follow the browser prompts to complete installation


## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Extension Not Loading">

    2. Clear browser cache
    3. Reinstall the extension
  </Accordion>
  
  <Accordion title="Login Issues">
    1. Clear browser cookies
    2. Check your credentials
    3. Verify network connectivity
