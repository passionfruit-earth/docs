**Chat** answers in plain language with citations; **Auto‑fill** writes into cells where supported. Tune [search configuration](/docs/fill-questionnaires/search-configuration) anytime to limit which sources we use.

1. Open Chat in the App or the Extension
2. Ask (one question, several at once, or follow-ups—see below)
3. Review the answer and citations
4. Open sources if you need detail
5. **Copy** or insert where supported

### Copy the answer

**Copy** the reply, then paste into Word, Excel, or your portal. See [Place answers](/docs/fill-questionnaires/place-answers) for inserting in the document.

### One question or several at once

**One question** → one focused answer. **Several questions in the same message** → one reply you can split across lines or cells (e.g. supplier packs with grouped rows).

### Follow-ups

In the **same chat**, ask again in plain English to reshape the last answer—citations stay tied to what we used.

Common follow-ups:
- **Table formatting**: ask to return the output as a table. For full table workflows, see [Create tables for questionnaires](/docs/fill-questionnaires/create-tables).
- **Language/tone**: request another language or a simpler/formal tone.
- **Combine/shorten**: merge prior replies or make the answer concise.

## Auto‑fill

Auto‑fill puts answers where you need them—e.g. [Excel functions](/docs/fill-questionnaires/excel-functions) for formulas in cells.
