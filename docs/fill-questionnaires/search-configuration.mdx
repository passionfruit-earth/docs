#### Overview

**Search configuration** is where you decide how Passionfruit searches your knowledge while you fill out a questionnaire. You can turn **Entity**, **Answer library**, and **Evidence** on or off, limit results to one or more companies or subsidiaries, and optionally pick individual files. The same settings apply to **every question** in that chat until you change them.

#### Select sources

**Select sources** defines which areas of your knowledge base are included in search: **Entity** (company-linked context and records), **Answer library**, and **Evidence**. Use the toggles to turn each type **on** or **off**. Only enabled sources are searched.

Your selection applies to the **whole questionnaire session**: once you set it, **every question** in that chat uses the same source mix until you change it. You do not need to reconfigure it for each question.

#### Filter on entities

Limit search to everything linked to one or more companies or subsidiaries. Only that evidence and those answers are used, so you don't mix information. In the configuration panel, select one or more entities and click **Apply**.

#### Manual selecting

Choose exactly which evidence files and answer library items to include. Answers then cite only those sources. In the configuration panel, tick the files or answer library items you want and click **Apply**. You can combine this with a filter on entities so the list shows only what's relevant.

#### Steps

1. Open **Chat** and set the questionnaire and entity (if needed).
2. Open **Select sources** and turn **Entity**, **Answer library**, and **Evidence** on or off. Optionally **filter by entities** or **manually select** specific evidence and answer library items.
3. Click **Apply** when your search configuration panel asks for it.
4. Ask your questions. The same source and scope settings apply to all of them until you change them.
5. Review answers and adjust configuration if needed, then continue.
