After you review, insert answers where you need them. In Chat you can **copy** the suggested text (including when you asked for a **table**); see [Ask questions](/docs/fill-questionnaires/ask-question).

## Options
- Copy to clipboard and paste
- Insert directly (where supported)
- For Excel, write to cells or use formulas
