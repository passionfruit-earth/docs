#### Overview
Save approved answers to your `Answer Library` from a conversation or from the review page so your team can quickly reuse consistent, vetted responses.


| Field | Description |
| --- | --- |
| Question | The questionnaire prompt the answer addresses. |
| Answer | The approved text that will be reused. |
| Entities | The entities this answer applies to (e.g., product, region, framework). |
| Evidence | The sources supporting the answer; used for citations and verification. |
| Notes | Internal context for maintainers; not inserted into responses. |

## Save from conversation
Save an approved answer directly during a conversation without breaking your flow.
Great for quickly capturing high‑quality responses as you work.

1. In Chat, open the suggested answer actions
2. Click `Save to library`
3. Set owner, review date, and tags
4. Save

## Save from review page
Approve and save from the detailed review view when you want to double‑check
citations and metadata before adding the answer to your library.

1. Open the answer detail page
2. Click `Approve & save` (or `Save to library`)
3. Set owner, review date, and tags
4. Save

## Tips
- Use clear titles and tags so others can find the answer
- Set a review date to keep content fresh
