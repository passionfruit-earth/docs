### Overview
Answer Library lets you quickly find and reuse approved answers across questionnaires and documents.
Search by text and refine with filters (tags, entities, owners, approval date, frameworks), then insert into `Answers`.

### App
Search by question text and answer content.

### Extension
Search and insert approved answers into your workflows.
