Your Answer Library is only reliable if answers stay accurate. Passionfruit catches many conflicts while you fill questionnaires, but manual entries can still drift. Use this monthly review to keep the library trustworthy.

## What Passionfruit checks automatically

At month end, Passionfruit scans for:

1. **Similar questions with different answers**
2. **Answers linked to documents that were updated**

Flagged items are marked `In Review`.

<Note>
  Passionfruit flags inconsistencies. Your team decides which answer is correct.
</Note>

## Why items get flagged

### Similar question, conflicting answer

If a manually added question-answer pair is very similar to an existing library item but the answer differs, both are flagged.

Example:
- "Do you have a quality policy?" -> `Yes`
- "Is a quality procedure defined?" -> `No`

### Linked evidence changed

If a certificate, policy, or specification linked to an answer is replaced with a newer version, that answer is flagged for review.

## What you receive

- Answers marked `In Review` in the Answer Library
- An email summary with:
  - Similar-question conflicts
  - Answers linked to updated documents
  - A direct link to review items

## Monthly review checklist

1. Open `Answer Library`
2. Filter by `In Review`
3. Open each flagged item and read the observation note
4. Either update the answer, or confirm it is still correct
5. Remove the review flag when done

Run this once per month. Do not skip it if you add answers manually.
