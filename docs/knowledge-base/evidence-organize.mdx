---
title: 'Organize evidence'
description: 'Review metadata, edit properties, and mark evidence ready for reuse'
---
Review and update extracted content and suggested metadata. Edit properties like `Title`, `Type`, `Source`, `Entities`, `Tags`, and `Owners`. Accept or adjust auto‑tag proposals, then approve to mark the item ready for reuse; all changes are recorded in the audit trail.

<Tabs>
    <Tab title="App" icon="browser">
    1. Go to `Evidence`
    2. Select an item to open details
    3. Edit properties: `Title`, `Type`, `Source`, `Entities`, `Tags`, `Owners`
    4. Review `Auto‑tags` proposals; accept or adjust
    5. Click `Approve` to mark ready for reuse (audit trail recorded)

    <img src="/images/evidence/EV-evidence-edit.png" className="w-full border-none h-auto my-0" alt="Edit evidence properties in App" />
  </Tab>

  <Tab title="Extension" icon="mobile">
    ### Edit evidence in details
    1. Open `Home` in the extension
    2. Click `Evidence` and select an item
    3. Use the edit controls to adjust properties
    4. Review inline suggestions under `Auto‑tags`; accept or change

    ### Approve evidence
    1. Click `Approve`
    2. Approval marks the item ready for reuse and records the action
  </Tab>
</Tabs>
