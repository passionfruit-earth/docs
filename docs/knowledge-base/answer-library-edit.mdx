#### Overview
Edit approved answers to update content, citations, owners, tags, and review dates. Changes are versioned.

<Tabs>
    <Tab title="App" icon="browser">
    1. Click `Answer Library`
    2. Select an answer and click `Edit`
    3. Update content, citations, owners, tags, and review date
    4. Click `Save` (version history is preserved across approvals)
  </Tab>
  <Tab title="Extension" icon="mobile">
    - You can view and insert approved answers
    - Editing happens in the App
  </Tab>
</Tabs>

#### Bulk edit answers (App)

In the main application, you can edit multiple answers at once. Select several answers in the Answer Library, then use the bulk edit action. Change one field while the rest stay locked, or update many answers in a single go.
