#### Overview
Edit approved answers to update content, citations, owners, tags, and review dates. Changes are versioned.

<Tabs>
    <Tab title="App" icon="browser">
    1. Click `Answer Library`
    2. Select an answer and click `Edit`
    3. Update content, citations, owners, tags, and review date
    4. Click `Save` (version history is preserved across approvals)
  </Tab>
  <Tab title="Extension" icon="mobile">
    - You can view and insert approved answers
    - Editing happens in the App
  </Tab>
</Tabs>

#### Bulk edit answers (App)

In the main application, you can edit multiple answers at once. Select several answers in the Answer Library, then use the bulk edit action. Change one field while the rest stay locked, or update many answers in a single go.

## Manage answer quality

### Monthly maintenance

Run this once per month to keep reused answers consistent:

1. Open `Answer Library`
2. Filter by `In Review`
3. Open each flagged item and read the observation note
4. Update the answer or confirm it is still correct
5. Remove the review flag

Items are typically flagged for one of two reasons:
- Similar questions have conflicting answers
- The linked document (certificate, policy, specification) was updated

Passionfruit flags inconsistencies. Your team decides what the correct answer is.

### Certification standards

Use this structure in your Answer Library so certification questions are answered consistently.

Keep these entries maintained:
1. **Which certificates do you have?**  
   Per site, include scheme name, accreditation body, expiry date, and linked certificate evidence.
2. **Are you BRC certified?**  
   `Yes/No`; if `Yes`, include accreditation body and expiry date.
3. **Are you IFS certified?**  
   `Yes/No`; if `Yes`, include accreditation body and expiry date.
4. **Which GFSI accreditation do you have?**  
   `Yes/No`; if `Yes`, include scheme, accreditation body, expiry date, and linked evidence.

When certificates renew, upload the new file, relink it, and update affected answer fields.
