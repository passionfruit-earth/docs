#### Overview
Edit entities to update fields, relationships, and attached evidence; changes are tracked.

<Tabs>
    <Tab title="App" icon="browser">
    1. Click `Entities`
    2. Select an entity and click `Edit`
    3. Update fields, relationships, and attached `Evidence`
    4. Click `Save` (changes are tracked in the activity panel)
  </Tab>
  <Tab title="Extension" icon="mobile">
    1. Go to `Home` in the extension
    2. Click `Entities`
    3. Select an entity and click `Edit`
    4. Update fields, relationships, and attached `Evidence`
    5. Click `Save`
  </Tab>
</Tabs>

#### Bulk edit entities (App)

In the main application, you can edit multiple entities at once. Select several entities in the list, then use the bulk edit action. Change one field while the rest stay locked, or update many entities in a single go.
