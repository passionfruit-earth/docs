Certifications appear in most supplier and customer questionnaires. Store them in a fixed structure so Passionfruit can answer consistently with the right dates, body, and evidence.

## Required question set

Every workspace should maintain these four Answer Library entries.

### 1) Which certificates do you have?

Use this as the master overview. Per site, include:
- Certificate name (for example: BRC, IFS, FSSC 22000)
- Accreditation body (for example: KIWA, SGS, Bureau Veritas)
- Expiry date
- Linked certificate document as evidence

### 2) Are you BRC certified?

- Answer `Yes` or `No`
- If `Yes`, include accreditation body and expiry date

### 3) Are you IFS certified?

- Answer `Yes` or `No`
- If `Yes`, include accreditation body and expiry date

### 4) Which GFSI accreditation do you have?

GFSI schemes include BRC, IFS, FSSC 22000, SQF, and others.

- Answer `Yes` or `No`
- If `Yes`, include:
  - Scheme name
  - Accreditation body
  - Expiry date
  - Linked certificate document as evidence

## Operating rule for renewals

When a certificate is renewed:

1. Upload the new certificate document
2. Link it to relevant certification answers
3. Update expiry date and any changed fields

Passionfruit will flag linked answers for review after document updates. Treat that as required maintenance, not optional cleanup.
