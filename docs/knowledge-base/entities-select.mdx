### Overview
Find and manage entities across your workspace.
Search by name and refine with filters (type, tags, owners, relationships).

### App
- **Search**: Search by name and fields.
- **Filters**: Filter by type, tags, owners, and relationships.

### Extension
- **Search**: Search by name.
- **Note**: Advanced filtering is available in the App.
