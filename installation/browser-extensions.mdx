## Chrome Extension

### Installation

1. Visit the [Chrome Web Store](https://chrome.google.com/webstore/detail/passionfruit)
2. Click "Add to Chrome"
3. Follow the browser prompts to complete installation

### Features

- Direct access to Passionfruit from your browser toolbar
- Answer questionnaires directly on web portals
- Save responses to your Answer Library
- Quick access to your knowledge base
- Real-time AI assistance while filling out forms

## Firefox Extension

### Installation

1. Visit the [Firefox Add-ons Store](https://addons.mozilla.org/en-US/firefox/addon/passionfruit)
2. Click "Add to Firefox"
3. Follow the browser prompts to complete installation

### Features

The Firefox extension offers the same features as the Chrome extension:
- Toolbar integration
- Direct questionnaire assistance
- Answer Library access
- Knowledge base integration
- AI-powered form filling

## Using the Extension

1. After installation, click the Passionfruit icon in your browser toolbar
2. Sign in with your Passionfruit account
3. Navigate to any questionnaire or form
4. The extension will automatically detect forms and offer assistance
5. Use the AI suggestions to quickly fill out responses
6. Save your answers to the Answer Library for future use

## Permissions

The browser extension requires the following permissions:
- Access to your Passionfruit account
- Access to web forms and questionnaires
- Storage for saved responses
- Network access for AI assistance

## Troubleshooting

If you encounter any issues:

1. Ensure you're signed in to your Passionfruit account
2. Check that the extension is enabled in your browser
3. Try refreshing the page
4. Clear your browser cache
5. Reinstall the extension if problems persist

For additional help, contact our [support team](mailto:hello@passionfruit.earth).

## Edge Extension

### Installation

1. Visit the [Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/passionfruit)
2. Click **Get**
3. Review permissions and click **Add extension**
4. Pin Passionfruit to your toolbar

## Enterprise Deployment

For organization-wide deployment:

### Chrome Enterprise

1. Download the enterprise package:
   ```bash
   https://static.passionfruit.earth/enterprise/chrome-extension.zip
   ```

2. Deploy using Group Policy:
   - Set up forced installation
   - Configure extension policies
   - Manage user permissions

### Firefox Enterprise

1. Download the enterprise MSI:
   ```bash
   https://static.passionfruit.earth/enterprise/firefox-addon.msi
   ```

2. Deploy using your MDM solution:
   - Configure automatic installation
   - Set organization policies
   - Manage permissions

### Edge Enterprise

1. Download the enterprise package:
   ```bash
   https://static.passionfruit.earth/enterprise/edge-extension.zip
   ```

2. Deploy using Intune or Group Policy:
   - Configure automatic installation
   - Set up extension policies
   - Manage access controls
