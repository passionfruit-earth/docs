## Microsoft Store Installation

1. Open Microsoft Excel
2. Navigate to the **Insert** tab
3. Click **Get Add-ins** or **Office Add-ins**
4. Search for "Passionfruit"
5. Click **Add** to install

<Note>
  This method requires administrator privileges on your computer. If you don't have admin rights, contact your IT department for assistance.
</Note>

## Manual Deployment

### For IT Administrators

1. Download the Excel manifest file:
   - [Excel Manifest](https://static.passionfruit.earth/manifests/excel-manifest.xml)

2. Deploy via Microsoft 365 Admin Center:
   - Go to **Settings** > **Integrated apps**
   - Select **Upload custom apps**
   - Choose the manifest file
   - Complete the deployment wizard

3. Configure deployment settings:
   - Select target users/groups
   - Set appropriate permissions
   - Enable automatic updates

## Using the Excel Add-in

1. Access Passionfruit from the **Home** tab
2. Click the **Passionfruit** button to open the sidebar
3. Log in with your Passionfruit credentials
4. Start using features:
   - Import questionnaire data
   - Export responses
   - Access the answer library
   - Generate AI-powered responses

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Add-in Not Loading">
    1. Verify Excel version (requires 2019 or later)
    2. Check internet connection
    3. Clear Office cache:
       - Close Excel
       - Delete contents of `%LOCALAPPDATA%\Microsoft\Office\16.0\Wef\`
       - Restart Excel
  </Accordion>
  
  <Accordion title="Installation Failed">
    1. Ensure you have admin rights
    2. Check system requirements:
       - Windows 10 or later
       - Office 365 or Excel 2019+
    3. Disable antivirus temporarily
  </Accordion>
  
  <Accordion title="Performance Issues">
    1. Update Excel to latest version
    2. Check available RAM (minimum 4GB recommended)
    3. Clear Excel cache and temporary files
  </Accordion>
</AccordionGroup>

### System Requirements

- Windows 10 or later
- Microsoft 365 or Excel 2019+
- 4GB RAM minimum
- Internet connection
- Admin rights for installation
